# bookjan-analysis — 리뷰 교차 분석 가이드

단순 리뷰 모아보기가 아닌, 여러 블로그의 패턴을 교차 분석하여 "진짜 분위기"를 추출하는 가이드.

## 핵심 철학

**개인 호불호가 일반화되면 안 된다.** 한 블로거가 "음악이 시끄럽다"고 했다고 그 장소의 단점이 되면 안 됨. 여러 블로거가 공통적으로 언급해야 비로소 장소의 특성으로 인정.

## 분석 파이프라인

```
1. blog_reviews에서 장소별 리뷰 수집 (is_ad=false만)
    ↓
2. 블로거 신뢰도 산정 (blogger_profiles)
    ↓
3. 신뢰도 가중치 적용하여 장단점 교차 분석
    ↓
4. 분류: 공통 특성 vs 개인 호불호
    ↓
5. 시간 기반 트렌드 분석
    ↓
6. place_insights에 저장
```

## 1단계: 블로거 신뢰도

| 요소 | 산정 기준 | 가중치 |
|------|----------|--------|
| 광고 비율 | 해당 블로거의 전체 리뷰 중 is_ad=true 비율 | 높을수록 감점 |
| 장단점 균형 | pros와 cons 모두 기술했는지 | 균형적이면 가점 |
| 구체성 | 메뉴명, 가격, 좌석 등 구체적 묘사 여부 | 구체적이면 가점 |
| trust_score | 기존 AI 판별 신뢰도 점수 | 직접 반영 |

신뢰도 = (1 - 광고비율) × 0.3 + 균형점수 × 0.3 + 구체성 × 0.2 + trust_score × 0.2

## 2단계: 장단점 교차 분석

같은 장소에 대한 여러 리뷰의 pros/cons를 비교. 유사한 키워드를 클러스터링.

**블로거 개인 사정 제외 원칙:** "가게가 바꿀 수 있는 것인가?"를 기준으로 판단.
- 제외: "거리가 멀다", "자주 방문하기 어렵다" (블로거의 위치 문제)
- 포함: "주차가 어렵다", "대중교통 접근성이 낮다" (가게 입지 특성)
- 개인 호불호: "개인 취향에 안 맞는 음악" (가게가 바꿀 수는 있지만 취향 문제)

예시:
```
리뷰A pros: ["조용한 분위기", "혼자 가기 좋음"]
리뷰B pros: ["정숙한 환경", "1인석 있음"]
리뷰C pros: ["시끄럽지 않음"]
→ 클러스터: "조용한 분위기" (3명 언급, 신뢰도 가중 평균 0.85)
```

## 3단계: 분류 기준

| 언급 블로거 수 | 분류 | JSON 키 |
|--------------|------|---------|
| 1명 | 개인 호불호 | personal_opinions |
| 2명 | 소수 의견 | personal_opinions (라벨: "소수") |
| 3명+ | 공통 특성 | common_pros / common_cons |
| 전체의 70%+ | 핵심 특성 | common_pros / common_cons (라벨: "핵심") |

## 4단계: 시간 기반 트렌드

- 같은 장소의 리뷰를 시간순 정렬
- 최근 6개월 리뷰 vs 이전 리뷰 비교
- 새로 등장한 장단점, 사라진 장단점 추적
- 변화가 감지되면 trend_changes에 기록

예시:
```json
{
  "trend_changes": [
    "2025년 이후 '가격 인상' 언급 3건 증가",
    "2026년 메뉴 리뉴얼 후 '메뉴 다양해짐' 언급 시작"
  ]
}
```

## 5단계: place_insights 저장

```json
{
  "place_id": "uuid",
  "analysis_version": 1,
  "common_pros": [
    {"text": "조용한 분위기", "count": 6, "weighted_score": 0.85, "label": "핵심"}
  ],
  "common_cons": [
    {"text": "좌석이 좁음", "count": 3, "weighted_score": 0.72, "label": "공통"}
  ],
  "personal_opinions": [
    {"text": "와인 종류가 적음", "count": 1, "type": "con", "label": "개인"}
  ],
  "trend_changes": ["..."],
  "overall_vibe": "조용하고 혼자 책 읽기 좋은 곳. 좌석은 다소 좁지만...",
  "confidence_level": 0.82,
  "analyzed_at": "2026-03-30T00:00:00Z"
}
```

## Phase 4 맞춤형 추천 (미래)

사용자 입력: "혼자 조용히 책 읽고 싶어, 와인 마시면서"
→ common_pros에 "조용한", "혼자" 포함 + 메뉴에 와인 있는 장소 매칭
→ confidence_level 높은 순으로 추천
